# 🖥️ Aplicación de Consola — Módulo 3

> **Fundamentos de Programación en JavaScript**
> Unidad solicitante: Equipo de desarrollo de una plataforma de aprendizaje interactivo

---

## 📍 Situación Inicial

Un equipo de desarrollo ha sido encargado de crear una aplicación en JavaScript que permita a los usuarios realizar operaciones básicas y trabajar con estructuras fundamentales del lenguaje.

Se busca desarrollar habilidades en JavaScript a través de una práctica aplicada, implementando los conceptos fundamentales del lenguaje.

---

## 🎯 Objetivo

Desarrollar una aplicación en consola que permita a los usuarios:

- ✅ Realizar operaciones matemáticas básicas.
- ✅ Implementar estructuras condicionales y de bucles.
- ✅ Usar funciones para modularizar el código.
- ✅ Trabajar con arreglos y objetos.

El código debe estar estructurado de manera clara según las mejores prácticas de desarrollo en JavaScript.

---

## 📋 Requerimientos

### Generales

- [ ] Debe ejecutarse en la consola del navegador.
- [ ] Debe permitir al usuario ingresar datos mediante `prompt` o variables predefinidas.
- [ ] Debe realizar al menos **tres operaciones diferentes** usando funciones.
- [ ] Debe utilizar **condicionales** y **estructuras de repetición**.
- [ ] Debe incluir **arreglos** y **objetos** para almacenar y manipular datos.

### Técnicos

| Requerimiento                  | Detalle                                                            |
| ------------------------------ | ------------------------------------------------------------------ |
| **Funciones y modularización** | Separación del código en funciones reutilizables                   |
| **Estructuras de control**     | Implementación de `if`, `switch`, `for`, `while` según corresponda |
| **Arreglos y objetos**         | Manipulación de datos utilizando estas estructuras                 |
| **Validaciones**               | Control de entradas del usuario para evitar errores                |

## 🚀 Cómo Ejecutar

1. Abre cualquier navegador (Chrome, Firefox, Edge).
2. Presiona `F12` o clic derecho → **Inspeccionar** para abrir las herramientas de desarrollador.
3. Ve a la pestaña **Console**.
4. Copia y pega el contenido de `app.js`, o cárgalo mediante un `<script>` en una página HTML.
